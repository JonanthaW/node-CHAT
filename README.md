## :rocket: Sobre a página

Aplicativo de sala de chat simples, igual ao antigo e popular serviço do MIRC e as salas de bate-papo dos antigos provedores de internet.

#### :wrench: Techs utilizadas:
* _HTML_
* _JS_
* _CSS_
* _NODEJS_
  * _EXPRESS_
  * _SOCKET.IO_
